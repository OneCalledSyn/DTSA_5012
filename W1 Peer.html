Problem #1: Simulate ANCOVA Interactions¶

In this problem, we will work up to analyzing the following model to show how interaction terms work in an ANCOVA model.

$$Y_i = \beta_0 + \beta_1 X + \beta_2 Z + \beta_3XZ + \varepsilon_i$$

This question is designed to enrich understanding of interactions in ANCOVA models. There is no additional coding required for this question, however we recommend messing around with the coefficents and plot as you see fit. Ultimately, this problem is graded based on written responses to questions asked in part (a) and (b).

To demonstrate how interaction terms work in an ANCOVA model, let's generate some data. First, we consider the model

$$Y_i = \beta_0 + \beta_1 X + \beta_2 Z + \varepsilon_i$$

where $X$ is a continuous covariate, $Z$ is a dummy variable coding the levels of a two level factor, and $\varepsilon_i \overset{iid}{\sim} N(0,\sigma^2)$. We choose values for the parameters below (b0,...,b2).

1. (a) What happens with the slope and intercept of each of these lines?¶

In this case, we can think about having two separate regression lines--one for $Y$ against $X$ when the unit is in group $Z = 0$ and another for $Y$ against $X$ when the unit is in group $Z = 1$. What do we notice about the slope of each of these lines?

The slopes of each of the lines are forced to be equivalent by the structure of the model; each are governed solely by $\beta_1$ regardless of the value of the dummy variable Z. The intercept is different for each value of the dummy variable Z: When $Z = 0$, the intercept is given by just $\beta_0$, but when $Z = 1$ the intercept is given by $\beta_0 + \beta_2 $. With no interaction term present, we will see the same slope but different intercepts (for non-zero values of $\beta_2$).

1. (b) Now, let's add the interaction term (let $\beta_3 = 3$). What happens to the slopes of each line now?¶

The model now is of the form:

$$Y_i = \beta_0 + \beta_1 X + \beta_2 Z + \beta_3XZ + \varepsilon_i$$

where $X$ is a continuous covariate, $Z$ is a dummy variable coding the levels of a two level factor, and $\varepsilon_i \overset{iid}{\sim} N(0,\sigma^2)$. We choose values for the parameters below (b0,...,b3).

In this case, we can think about having two separate regression lines--one for $Y$ against $X$ when the unit is in group $Z = 0$ and another for $Y$ against $X$ when the unit is in group $Z = 1$. What do you notice about the slope of each of these lines?

With an interaction term added into the mix, we will now see a visual disparity in both the slope and the intercept of the regression lines. As before, when $Z = 0$, the intercept is given by just $\beta_0$ and when $Z = 1$ the intercept is given by $\beta_0 + \beta_2 $. The difference now is that when $Z = 0$, the slope is given by $\beta_1$ but when $Z = 1$ the slope is given by $(\beta_1 + \beta_3)$. This means that as long as $\beta_3$ is non-zero, the slopes of the two lines will not be exactly the same numerically.

In part b, we discovered that there was a difference of about 7.25 in the mean MPG between the group of manual transmission cars and automatic transmission cars. By examining the values of the regression lines' intercepts (appears to be ~7 MPG gap visually) and the distance between the two parallel regression lines, it becomes evident that there is a mean difference in MPG approximately equal to the result calculated prior. In part c, we examine the F-test results for the full model that included the interaction term vs the reduced model with the interaction term removed, and concluded that the reduced model was sufficient based on the p-value, meaning that the interaction term was not significant. By looking at the slopes of the regression lines in the interaction plot, we notice that they look almost exactly the same as the regression lines without the interaction term - parallel. This is because the effect the interaction term has on the slope is so miniscule that the change is impercetible. Further investigation shows the parameter estimate for the interaction term to be ~0.0004, which is over 100 times smaller than the other slope parameter estimate, so numerically we would assume that it would be dominated.
